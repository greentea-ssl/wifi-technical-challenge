ブロードキャストアドレス 192.168.4.255、ポートは 50000 + id


## AIでの処理

matcha_receiver/scripts/uplink_receiver.pyにUDPの受信&topicへのpublishが実装されている。
- データの形式: `UplinkData.msg`
- topicの名前: `bluenegative/uplink_data/robot_{i}` (i: robotのid)


---

> **HID 経由 CAN テレメトリ転送 (Port 51000+id, JSON)** は本書ではなく [hid_bridge.md](./hid_bridge.md) を参照。CAN 上のテレメトリ用メッセージ種別 (`11111`) を HID が JSON 化して PC へブロードキャストする診断用チャネル (v2.0.0 で導入)。
